# Victor Li — Portfolio

A single-page portfolio targeting one company, showcasing software engineering projects as in-depth case studies (what was built, what was learned, mistakes made).

## Stack

TypeScript · Next.js 16 · React 19 · Tailwind CSS v4 · deployed on Vercel.
Follow UI/UX Pro skill recommendations for design decisions.

## Design direction

Clean, elegant, interactive. Favour whitespace, subtle motion, and purposeful detail over decoration.

## Structure

- **Hero** — name, GitHub/LinkedIn links, resume expand-from-icon
- **Projects** — each project is a case study: problem, what was built, lessons learned, mistakes

## Dev

```bash
npm run dev   # http://localhost:3000
npm run build # verify before pushing
```

Vercel deploys automatically on push to `main`.
